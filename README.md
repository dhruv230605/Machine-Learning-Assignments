# Machine Learning Projects

A collection of machine learning and data analysis work focused on building end-to-end ML pipelines, statistical analysis, and evaluation of classical machine learning models across diverse data modalities.

The repository emphasizes clean preprocessing, feature engineering, reproducible experimentation, and metric-driven evaluation.

---

## Project Overview

This repository includes work involving:

- Multimodal learning using audio and image data  
- Tabular data analysis and statistical learning  
- Feature extraction, normalization, and exploratory analysis  
- Model evaluation using accuracy, F1-score, and class-wise metrics  

---

## Key Concepts & Techniques

- End-to-end ML pipeline design  
- Feature engineering and preprocessing  
- Classical ML models (classification & statistical learning)  
- Metric-driven evaluation and error analysis  
- Data visualization for insight generation  
- Reproducible experimentation
